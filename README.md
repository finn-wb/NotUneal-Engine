# NotUneal Engine

A simplified game engine inspired by Unreal, featuring a complete editor, packaging system, asset management, and extensible renderer.

## Features

- **Editor**: Visual editor for scene composition and project management
- **Packaging System**: Build and package games with all assets
- **Asset Management**: Import and organize meshes, textures, materials, and shaders
- **Game Code**: User projects compiled by editor with engine API bindings
- **GameObject System**: Universal actor/component system for all scene objects
- **Basic Components**: Transform, Mesh, Light, Camera, and custom components
- **Renderer**: Extensible rendering pipeline with custom shader support
- **Visual Shader Editor**: Simplified visual scripting for shader creation

## Project Structure

```
NotUneal-Engine/
├── Engine/              # Core engine library
├── Editor/              # Desktop editor application
├── Runtime/             # Runtime libraries for packaged games
├── Tools/               # Utility tools and asset importers
├── Examples/            # Example projects
└── Documentation/       # Engine documentation
```
